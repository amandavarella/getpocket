# Get Pocket
Helpers to manage pocket


## Libraries
```
gem install dotenv
```

## Environment
Add a .env file with the following keys

```
ENVATO_CONSUMER_KEY=
ENVATO_ACCESS_TOKEN=
GOOGLE_CONSUMER_KEY=
GOOGLE_ACCESS_TOKEN=
```
